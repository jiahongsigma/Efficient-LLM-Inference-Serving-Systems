# 中文翻译 · Chinese translation

欢迎把这门课程翻译成中文！Translations of this course into Chinese are warmly welcome.

翻译文件放在本目录下，文件名与英文原文保持一致。仓库根目录的英文文件是**权威来源**，翻译请随之更新。
Translated files live here with the same names as the English source at the repo root, which is the
**source of truth** — keep translations in sync as it changes.

## 如何参与 · How to help

1. 先开一个 **Translation help (翻译)** issue 认领某个文件，避免两个人翻译同一篇。
   (Open a *Translation help* issue to claim a file so two people don't translate the same one.)
2. **只翻译正文，不翻译机器部分** — 代码、文件名/标识符、数学公式、以及术语（KV cache、roofline、
   prefill/decode、TTFT/TPOT）保留英文，必要时首次出现时附中文解释。
   (Translate prose only; keep code, identifiers, math, and terms-of-art in English.)
3. **保持结构一致**（相同的标题与 `§N.x` 编号），方便交叉引用与后续同步。
   (Mirror the structure so readers can cross-reference and it stays easy to sync.)
4. 部分翻译也很好 —— 翻多少算多少，并注明停在哪里。
   (A partial translation is fine — do what you can and note where you stopped.)
