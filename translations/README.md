# Translations

Community translations of the course. Each language lives in its own subdirectory named by its
[ISO 639-1 code](https://en.wikipedia.org/wiki/List_of_ISO_639-1_codes), with files mirroring the
English source (same names, same structure). The English files at the repo root are the
**source of truth**; translations track them.

| Language | Directory | Status |
|---|---|---|
| 简体中文 · Chinese (Simplified) | [`zh/`](zh/) | started — help wanted |

Want to add a language? Open a **Translation help (翻译)** issue (New issue → the dropdown), then
create `translations/<code>/` with a `README.md` like [`zh/README.md`](zh/README.md). See the
**Translations** section of [`../CONTRIBUTING.md`](../CONTRIBUTING.md) for conventions.

**Translate the prose, not the machinery** — keep code, file/identifier names, math, and terms-of-art
(KV cache, roofline, prefill/decode, TTFT/TPOT) in English; gloss them on first use if it helps.
